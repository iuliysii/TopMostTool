# Changelog

All notable changes to this project will be documented in this file.
This project follows a pragmatic changelog format.

## [1.0.0] - 2026-03-12

### Added
- Global hotkey to toggle top-most state of the active window.
- System tray menu with pinned windows list and quick actions.
- One-click “clear all top-most” action.
- Autostart toggle via Windows registry.
- Build support with PyInstaller.

### Notes
- Requires administrator privileges on Windows for global hooks and window management.
