# Release Notes — TopMost Tool 1.0.0

**Release date:** 2026-03-12

## Highlights
- Global hotkey to toggle top-most state for the active window.
- Tray menu to view pinned windows and manage actions quickly.
- One-click clear-all and autostart toggle.

## What’s Included
- **Hotkey toggle**: default `Ctrl+Space`.
- **Tray menu**: list pinned windows, unpin individually, clear all, exit.
- **Autostart**: stored under `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`.

## Requirements
- Windows 10/11
- Administrator privileges (required for global hook and window manipulation)

## Install / Run
- **EXE**: run `TopMostTool.exe` (recommended to launch as Administrator).

## Notes
- Tray menu displays window titles; this may expose sensitive info in shared environments.

## Known Issues
- None reported for this release.
